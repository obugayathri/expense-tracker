# Personal Expense Tracker 💰

## 📌 Project Overview
This is a simple and user-friendly Expense Tracker web application. It helps users to manage their personal finances by tracking income and expenses.

---

## 🚀 Features
- Add income and expense transactions
- View total income, total expenses, and balance
- Category-wise expense visualization using Pie Chart
- Transaction history with delete option
- Data stored using LocalStorage (persists after refresh)
- Input validation for better user experience

---

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

---

## 📊 How It Works
- User enters transaction details (title, amount, category, type)
- Data is stored in LocalStorage
- UI updates dynamically without page reload
- Chart shows expense distribution

---

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
